License and Permissioning

Data is in the public domain, as noted here: https://data.sfgov.org/Public-Safety/Fire-Department-Calls-for-Service/nuek-vuh3/data

Data Set Information

The data set is about 11,000 rows. We accessed this data on April 19, 2020.

March 2020 Potentially Life Threatening Incidents across neighborhoods, grouped by response type

Data Summary

We analysed incidents reported as "Potentially Life Threatening" over March 2020, across the top 20 neighborhoods of San Francisco in terms of incidents reported. We compared Code 2 responses in the "Potentially Life Threatening" category to all other response types, because Code 2 was by far the most common. Code 2 is classified as a routine call, and not an emergency response that necessarily requires urgent attention. Interestingly, most of the calls in the "Potentially Life Threatening" categorization for this dataset also doesn't require emergency response, which sounds counterintuitive at first.

Data Wrangling

We removed neighborhoods that had number of incidents outside the top 20 to make the visualizations easier to understand. We perform additional data wrangling in Javascript at runtime to manipulate the data into a hierarchical format acceptable by D3's stratify function.

Conclusions

About half of all "Potentially Life Threatening" calls in March were of the Code 2 response type, regardless of neighborhood. Certain areas, such as the Tenderloin, are more prone to incidents deemed "Potentially Life Threatening".

Interactivity

We added tooltips on mouseover to each visualization. The tooltips displays the count of each incident in March.
